# BuildOne Website Features

## New Design
- Clean, minimal design inspired by Claude AI
- Removed cartoony elements and excessive animations
- Professional, modern appearance
- Responsive design for all screen sizes

## Projects Section
- New Projects page accessible from main navigation
- Two featured projects:
  - **The Rusty LightHouse**: Lighthouse management system
  - **Rusty-Developer-Studio**: Development environment

## Project Detail Pages
- Individual pages for each project (`/projects/rusty-lighthouse`, `/projects/rusty-developer-studio`)
- Editable content with "Edit Content" button
- Article-style layout with:
  - Editable title
  - Image upload with rounded corners and borders
  - Rich text content area
  - Auto-save to localStorage

## Settings & Customization
- Comprehensive Settings page (`/settings`)
- **Appearance Options:**
  - Light/Dark theme toggle
  - Color themes: Blue, Green, Purple, Orange
  - Font size: Small, Medium, Large
- **Accessibility Features:**
  - High contrast mode
  - Reduced motion for users with vestibular disorders
- Settings persist across browser sessions

## Theme System
- CSS variables for consistent theming
- Automatic theme application on page load
- Smooth transitions between themes
- Dark mode support throughout the site

## Technical Improvements
- Modern React patterns with hooks
- Utility-based theme management
- Local storage for user preferences
- Clean, maintainable CSS architecture
- Mobile-responsive design

## Usage
1. Navigate to Projects to see available projects
2. Click on any project to view/edit its details
3. Use Settings to customize appearance and accessibility
4. All changes are automatically saved locally
